## üí° Project Overview

**NoiseScope** is a Minimum Viable Product (MVP) for a crowdsourced system designed to collect, visualize, and analyze real-time ambient noise data. The goal is to create detailed, up-to-date noise maps of urban areas, helping city planners, researchers, and citizens understand and address noise pollution.

This repository contains the front-end (HTML/CSS/JS) for the initial access portal, which directs users to either the **Admin Dashboard** or the **User/Contributor Portal**.

## ‚ú® Key Features

* **Dual Access Portal:** Secure entry points for both Administrators and Contributors.
* **Contributor Portal:** Allows users to submit noise level (dB) and location data via their mobile devices.
* **Admin Dashboard:** Provides tools for data review, noise map visualization, and user management (future development).
* **Simple, Responsive Design:** Ensures accessibility on various devices.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, Safari) to view the front-end pages.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Varun987-a11/NoiseScope.git](https://github.com/Varun987-a11/NoiseScope.git)
    cd NoiseScope
    ```

2.  **Open the Access Portal:**
    Simply open the `index.html` file in your preferred web browser.

    ```bash
    # Example command (may vary by OS)
    open index.html 
    ```

3.  **Navigate:**
    * Click **ADMIN** to be directed to `admin_login.html`.
    * Click **USER / CONTRIBUTOR** to be directed to `user_login.html`.

**(Note: This MVP currently only includes the HTML structure. The actual login and data submission functionality would require backend integration.)**
